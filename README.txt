            "How to Run the Program"
1) We implemented the project using HTML,CSS,and Javasscript.
2) In order to run the project, go to "index.html" file to open the webpage  and click on the "blue circle" that is located at the bottom right side. 
3)When clicked on the blue circle, the chatbot will appear for the user.
4) The user have to click on the given questions asked on the chatbot.
